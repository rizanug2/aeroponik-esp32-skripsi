# Sistem Aeroponik ESP32

Project tugas akhir:
Rancang Bangun Sistem Regulasi Nutrisi Adaptif pada Aeroponik Kentang Berbasis ESP32

## Deskripsi
Sistem ini digunakan untuk mengontrol pH dan EC secara otomatis menggunakan ESP32.

## Fitur
- Monitoring pH & EC
- Kontrol nutrisi otomatis
- Integrasi Firebase
